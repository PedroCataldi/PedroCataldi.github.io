This talk will provide a practical introduction to working with IllustrisTNG and analysing cosmological simulation data using Python. Using a set of analysis tools developed for galaxy studies, I will show common workflows for data analysis, kinematic and morphological decomposition, radial profiles, and visualisation. The goal is to provide participants with practical examples and scripts that can serve as a starting point for their own research projects.
